# Example: PDF Reading Session

User prompt:

```text
Use $daily-work-planner to plan a 14:00-18:00 reading session.
Goal: read two PDFs and create a 1500-word presentation outline.
Hard deadline: 18:00.
Files: paper-a.pdf, paper-b.pdf, notes.docx.
```

Expected plan shape:

- Planning budget no more than 8 minutes
- File priority: main PDFs first, notes as reference
- Soft deadline around 17:00-17:20
- Milestones for scan, priority reading, outline drafting, final check
- Minimum deliverable: a complete outline with core claims
- Delay response: skip optional sections and preserve final outline
