# Example: Word Revision Session

User prompt:

```text
Use $daily-work-planner to plan a 3-hour document revision session.
Goal: revise my report and prepare the final Word version.
Hard deadline: 21:00.
Files: report.docx, rubric.pdf, previous-feedback.txt.
```

Expected plan shape:

- Planning budget no more than 8 minutes
- File priority: report.docx first, rubric second, feedback third
- Milestones for issue scan, priority revision, formatting, export check
- Acceptance criteria for each revision block
- Minimum deliverable: corrected main report with final filename saved
- Delay response: cut polish before cutting correctness
