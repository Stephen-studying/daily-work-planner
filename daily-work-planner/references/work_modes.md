# Work Modes

Use the mode that best matches the deliverable. If multiple modes apply, use Mixed Work Mode and protect the final deliverable first.

## Deep Writing Mode

Use for papers, reports, essays, proposals, applications, manuscript sections, and long-form writing.

Focus on:

- Section structure
- Claim sequence
- Paragraph function
- Citation gaps
- Figure/table consistency
- Draft completeness
- Submission readiness

Minimum deliverable examples:

- A complete outline with section claims
- A rough but complete first draft
- A revised version of the highest-risk sections

## Fast Reading Mode

Use for PDFs, papers, textbooks, policy files, technical docs, and course materials.

Focus on:

- Reading purpose
- Must-read sections
- Skippable sections
- Key claims or formulas
- Notes that feed the next task

Minimum deliverable examples:

- One-page structured notes
- Five core takeaways
- A list of pages or sections requiring deeper reading

## Exam Review Mode

Use for exams, memorization, problem practice, and last-mile review.

Focus on:

- High-frequency topics
- Weak topics
- Problem types
- Recitation loops
- Error review
- Testable output

Minimum deliverable examples:

- Top topics list
- One practice set completed and checked
- A short-answer memory sheet

## Code Development Mode

Use for feature work, debugging, refactoring, tests, and technical validation.

Focus on:

- Target behavior
- Relevant files
- Minimal runnable version
- Test or verification command
- Debug order
- Risky dependencies

Minimum deliverable examples:

- A runnable implementation
- A failing test reproduced and diagnosed
- A fix plus a focused verification command

## PPT Production Mode

Use for slides, talks, defenses, group meetings, and project presentations.

Focus on:

- Talk length
- Slide count
- Storyline
- Required figures
- Visual consistency
- Speaker notes
- Minimum presentable deck

Minimum deliverable examples:

- Title, background, core results, conclusion
- A rough deck with complete slide titles
- A final deck with export check

## Document Revision Mode

Use for Word documents, reports, forms, formatting, polishing, and final submission checks.

Focus on:

- Main document
- Formatting rules
- Heading hierarchy
- Figures and tables
- References
- Export and filename checks

Minimum deliverable examples:

- Corrected main content
- Passing format checklist
- Final export-ready file

## Data Organization Mode

Use for spreadsheets, lab data, images, references, and file cleanup.

Focus on:

- Source files
- Naming rules
- Table structure
- Traceability
- Missing values
- Final folder or workbook layout

Minimum deliverable examples:

- Cleaned primary table
- Organized source folder
- Summary sheet with known issues

## Research Planning Mode

Use for research questions, methods, experiments, evidence mapping, and analysis planning.

Focus on:

- Research question
- Evidence need
- Method or protocol
- Data/source priority
- Next research artifact

## Literature Review Mode

Use for citation screening, related work, reference sorting, and evidence synthesis.

Focus on:

- Review scope
- Inclusion/exclusion criteria
- Must-read sources
- Citation gaps
- Claim-to-reference mapping

## Application Materials Mode

Use for resumes, CVs, personal statements, portfolios, scholarships, and applications.

Focus on:

- Target requirements
- Required files
- Core story
- Deadline and submission checks
- Final package readiness

## Translation and Polishing Mode

Use for translation, bilingual drafts, academic polishing, localization, and style cleanup.

Focus on:

- Source scope
- Target language/style
- Terminology
- Meaning preservation
- Consistency check

## Meeting Preparation Mode

Use for meetings, interviews, agendas, briefings, and follow-up planning.

Focus on:

- Objective
- Decision points
- Materials to prepare
- Questions
- Follow-up actions

## Lab Work Mode

Use for experiments, protocols, samples, instruments, and lab records.

Focus on:

- Protocol readiness
- Sample/material checklist
- Instrument or measurement steps
- Data recording
- Backup and traceability

## Admin Task Mode

Use for forms, emails, submissions, appointments, receipts, and paperwork.

Focus on:

- Requirements
- Recipients
- Attachments
- Submission confirmation
- Follow-up reminders

## Design and Creative Mode

Use for design briefs, mockups, prototypes, visual assets, UI/UX, and creative work.

Focus on:

- Audience
- Constraints
- Concept draft
- Handoff format
- Export/share readiness

## Mixed Work Mode

Use when the task combines reading, writing, coding, data, and presentation work.

Focus on:

- Final deliverable
- File priority
- First irreversible blocker
- Minimum deliverable
- Checkpoint-based rescheduling
